# References

External published papers/reports backing the project's economics and hydrology
methodology (as opposed to `../v1_*`, `../v2_*`, `../v3_*`, which are this project's own
write-ups).

## Core cited references
- **`gourevitch-gold-garcia_wetlands-flood-mitigation.pdf`** — Gourevitch, Gold & Garcia,
  wetland flood-mitigation ecosystem-service valuation.
- **`taylor-druckenmiller-2022_wetlands-flooding-clean-water-act.pdf`** — Taylor &
  Druckenmiller (2022), wetland loss under the Clean Water Act and its effect on flooding
  (uses NFIP claims data). The team's Box folder (`replication_files/`) has what looks
  like this paper's own replication package (NFIP claims/policies, wetland-extent GEE
  scripts, county/zip panel) if the underlying data is ever needed — not committed here.

## Hydrology
- `assessing-runoff-sensitivity.pdf` — runoff sensitivity to land cover/precipitation.
- `satellite-imaging-population-flood-exposure.pdf` — satellite-based estimates of
  population growth in flood-exposed areas.

## Economics / insurance / financing
- `cooperative-game-upstream-downstream-flood-risk.pdf` — game-theoretic cost allocation
  for flood-risk prevention across upstream/downstream river basins.
- `bgr-stormwater-fees-2017-report.pdf` — stormwater utility fee structures.
- `community-based-catastrophe-insurance.pdf` — community-based catastrophe insurance
  models.
- `improving-flood-resilience-insurance-nature-based-mitigation.pdf` — combining
  insurance mechanisms with nature-based mitigation.
- `parametric-insurance-for-disasters-2020.pdf` — parametric (index-based) disaster
  insurance.
